Cal-Poly-ME-405-Term-Project
=======================================

Python code and term project description for Cal Poly's Mechatronics final term project, organized and written by Jireh Velasquez and Aubrey Perez. This project included the programing of Pololu's Romi robot to run through an obstacle course track in a fast and reliable manner.
